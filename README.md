# Smart Career Path Recommender System

A Python-based AI system that recommends optimal career paths for students based on their skills, interests, and education level.

## Features

- **Student Profile Management**: Create and manage student profiles
- **AI Career Matching**: Machine learning-based career recommendations
- **Learning Path Generation**: Personalized skill development roadmaps
- **NLP Analysis**: Sentiment analysis and text processing
- **Data Visualization**: Interactive charts and graphs
